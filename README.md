This is my personal portfolio website built with React, TypeScript, Vite, Tailwind CSS and Framer-Motion. It showcases selected projects, technical skills, and experience as a full-stack developer.

## Project setup

1. Clone the repository.
2. Install dependencies:
   - `npm install`
3. Run the server:
   - `npm run dev`

The app runs on port 5173

## Production Build

1. Create a production build:
   - `npm run build`
2. Preview locally:
   - `npm run preview`

## Deployment

This project can be deployed using platforms such as Vercel, Netlify, or Render.
